# Hyperopt Skill

**Description:** Hyperparameter optimization using Optuna Bayesian optimization.

## Capabilities
- Bayesian hyperparameter optimization
- Multi-objective optimization
- Pruning of unpromising trials
- Parallel optimization support

## Tools
- `hyperopt_optimize`: Optimize model hyperparameters
- `hyperopt_multi_model`: Optimize multiple models

## Tags
hyperparameter, optimization, optuna, bayesian, tuning
