# n8n Workflow Integration

Trigger and manage n8n workflows as Jotty skills.

## Description

This skill integrates with n8n workflow automation to list, trigger, monitor, and manage workflows. Each n8n workflow is also registered as a derived skill for direct discovery by the planner.

## Type
base

## Capabilities
- automation
- workflow-trigger

## Use When
User wants to trigger an n8n workflow, list automations, check execution status, or manage workflow activation

## Triggers
- "n8n workflows"
