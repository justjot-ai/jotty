# Model Metrics Skill

**Description:** Comprehensive model evaluation metrics for classification, regression, and ranking.


## Type
derived

## Base Skills
- data-profiler

## Capabilities
- Classification metrics (accuracy, F1, AUC, precision, recall)
- Regression metrics (MSE, RMSE, MAE, R2, MAPE)
- Cross-validation scoring
- Confusion matrix analysis
- Threshold optimization
- Calibration analysis

## Tools
- `metrics_classify_tool`: Classification metrics
- `metrics_regress_tool`: Regression metrics
- `metrics_crossval_tool`: Cross-validation metrics
- `metrics_calibration_tool`: Probability calibration

## Dependencies
- scikit-learn
- numpy
- pandas

## Tags
metrics, evaluation, classification, regression, cross-validation
