# Dimensionality Reduction Skill

**Description:** Dimensionality reduction techniques for visualization and feature extraction.


## Type
derived

## Base Skills
- data-profiler

## Capabilities
- Principal Component Analysis (PCA)
- t-SNE visualization
- UMAP embedding
- Feature variance analysis
- Explained variance calculation

## Tools

- `tsne_reduce_tool`: t-SNE for visualization
- `umap_reduce_tool`: UMAP embedding
- `variance_analysis_tool`: Feature variance analysis

## Dependencies

- numpy
- umap-learn (optional)
