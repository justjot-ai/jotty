# Feature Tools Skill

**Description:** Automated deep feature synthesis using Featuretools library.

## Capabilities
- Automated feature generation from relational data
- Deep feature synthesis (DFS)
- Feature primitive discovery
- Entity set management
- Time-based feature generation

## Tools
- `featuretools_dfs_tool`: Deep feature synthesis
- `featuretools_primitives_tool`: Apply feature primitives
- `featuretools_normalize_tool`: Normalize entity data

## Dependencies
- featuretools
- pandas
- numpy

feature-engineering, dfs, featuretools, automated-ml, relational
