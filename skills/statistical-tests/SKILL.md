# Statistical Tests Skill

**Description:** Statistical hypothesis testing and analysis.

## Capabilities
- T-tests (one-sample, two-sample, paired)
- ANOVA (one-way, two-way)
- Chi-square tests
- Normality tests
- Correlation tests
- Non-parametric tests

## Tools
- `ttest_tool`: T-test for mean comparison
- `anova_tool`: Analysis of variance
- `chisquare_tool`: Chi-square test
- `normality_tool`: Normality tests
- `correlation_test_tool`: Correlation significance

## Dependencies
- scipy
- statsmodels
- numpy

## Tags
statistics, hypothesis-testing, ttest, anova, chi-square
