# Auto-sklearn Skill

**Description:** AutoML using auto-sklearn with ensemble selection.

## Capabilities
- Automated algorithm selection
- Hyperparameter optimization
- Ensemble construction
- Meta-learning warmstart
- Time-based budget control

## Tools
- `autosklearn_classify_tool`: AutoML classification
- `autosklearn_regress_tool`: AutoML regression
- `autosklearn_ensemble_tool`: Get ensemble info

## Dependencies
- auto-sklearn
- scikit-learn

## Tags
automl, auto-sklearn, ensemble, meta-learning, optimization
